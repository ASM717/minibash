# minishell

School 21 project. Implementation of own (mini)bash.

By [Gregory](https://github.com/Gregory-Unknown) && [Stepan](https://github.com/ASM717)

## Features
`pipes`, `$?`, redirections: `>` `>>` `<` `<<`

Builtins: `cd`, `pwd`, `echo`, `export`, `unset`, `env`, `exit`.

Signals: `Ctrl-C`, `Ctrl-\`, `Ctrl-D`

and much more..../
